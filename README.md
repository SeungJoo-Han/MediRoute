# MediRoute 🏥

**대학병원 셔틀버스 포함 길찾기 서비스**

출발지와 목적지 병원을 입력하면 셔틀버스 + 대중교통을 모두 고려해 최적 경로를 안내합니다.

## 주요 기능

- 셔틀버스 + 도보 / 셔틀버스 + 대중교통 복합 경로 계산
- ODsay API 연동 시 실제 대중교통 소요시간 및 요금 표시
- 날짜(요일)별 시간표 반영 (평일 / 토요일 / 일요일)
- 카카오 지도에 경로 시각화 (지하철·버스·셔틀·도보 구간별 색상)
- 출발지 주소 자동완성

## 기술 스택

- **Backend**: Python, FastAPI, Uvicorn
- **Frontend**: HTML / CSS / JavaScript (빌드 도구 없음)
- **지도**: 카카오 지도 JavaScript SDK
- **대중교통**: ODsay API (선택)
- **배포**: Docker / Koyeb

## 로컬 실행

### 1. 환경변수 설정

```bash
cp .env.example .env
```

`.env` 파일을 열어 API 키를 입력합니다.

| 변수 | 설명 | 발급처 |
|------|------|--------|
| `KAKAO_REST_API_KEY` | 서버 geocoding용 | [카카오 개발자센터](https://developers.kakao.com) |
| `KAKAO_JS_KEY` | 브라우저 지도용 | 동일 |
| `ODSAY_API_KEY` | 대중교통 실시간 경로 (선택) | [ODsay Lab](https://lab.odsay.com) |

### 2. 서버 실행

```bash
cd backend
pip install -r requirements.txt
python run.py
```

브라우저에서 `http://localhost:8000` 접속

## 배포 (Koyeb)

1. GitHub 레포 연결
2. Build method: **Dockerfile**
3. Port: `8000`
4. 환경변수 설정 (`.env`의 키들을 Koyeb 대시보드에 입력)
5. 카카오 개발자센터에서 배포 URL을 Web 플랫폼에 추가
